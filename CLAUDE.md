# PET Photon Simulator (Angular + NgRx Signal Store)

## Project Context
- **Lead Developer:** Ignacio Lemmo (CTO / Software Engineer)
- **Tech Stack:** Angular 19+, Tailwind CSS, NgRx Signal Store, RxJS.
- **Backend Reference:** `nuclear-api` (FastAPI / Python 3.12).
- **Goal:** Dashboard reactivo para visualización de física nuclear.

## Engineering Standards
- **State Management:** NgRx Signal Store (`signalStore`, `withState`, `withMethods`, `withComputed`).
- **Reactivity:** Uso extensivo de Angular Signals para el binding de UI.
- **Styling:** Tailwind CSS (Dark Mode, Utility-first).
- **Architecture:** Standalone Components y Functional Store.

## Commands
- **Install:** `npm install`
- **Dev:** `ng serve`
- **Build:** `ng build`
